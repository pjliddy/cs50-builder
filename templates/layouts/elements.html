Blockquotes

Blockquote

The <blockquote> element is used to present content from another source with an optional citation.

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.

Citation wrapped by a <footer> element inside <blockquote> tags

Reverse Blockquote

To right-align the quote, use the .blockquote-reverse class on a <blockquote> element.

Description List

Description List Element

The <dl> element indicates a description list. <dt> tag shows terms. <dd> tag shows descriptions.

Coffee
- black hot drink
Milk
- white cold drink

Horizontal Description List Class

The .dl-horizontal class lines up the description list side-by-side when the browser window expands:
