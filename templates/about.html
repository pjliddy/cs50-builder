About Bootstrap Theme Builder

This project is a Minimum Viable Product (MVP) that tests the concept of a visually-oriented tool for the configuration of Bootstrap themes. The primary user story for effectively testing the concept is:

As a designer, I want to configure Bootstrap visually because that is how I think about design patterns.

Additional user stories that support user testing of the proof of concept include:

As a potential user, I want to register so I can try out the theme builder.

As a user, I want to save my themes so I can use them later.

As a user, I want to load themes in my library so I can continue to iterate on my design.

Application

  • Jinja2 Templating Engine
  • SQLite3 Database Engine

Presentation

  • HTML5/CSS3/JavaScript
